# Binary – E-Learning Platform Interface

A modern e-learning platform interface built with Next.js and TypeScript, focused on delivering a clean and minimal learning experience.


## Overview

Binary is an application designed to simulate an online learning platform.

The project focuses on creating a structured and intuitive interface for exploring courses, viewing content and managing the learning experience.


## Core Features

- Course exploration with filtering system  
- Course details with structured content (overview, syllabus, requirements, reviews)  
- Course enrollment flow
- Wishlist functionality  
- Contact page  
- Multi-section homepage  


## Tech Stack

- Next.js  
- TypeScript  
- Redux Toolkit  
- Framer Motion
